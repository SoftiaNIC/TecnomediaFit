# Inventory and expenses

El inventario se limita a productos liquidos. El sistema tambien debe registrar venta de liquidos, gastos y obligaciones operativas.

## Inventario

- Solo liquidos.
- Venta de liquidos incluida en el MVP.
- Factura/recibo basico interno para venta de liquidos.
- Creditos de estudiantes para cubrir deudas pendientes de inventario.

## Gastos

- Gastos fijos.
- Gastos variables.
- Impuestos con tarifa fija.
- Limpieza.
- Luz como costo variable.
- Alquiler del lugar.
- Imprevistos.

## Caja

- Diferenciar caja chica de cuenta principal.

## Dudas

- Confirmar si liquidos tienen precio de venta, costo interno, stock minimo o combinacion de estos datos.
- Confirmar formato de factura/recibo basico para venta de liquidos.
- Definir si gastos pendientes tienen fecha de vencimiento y responsable.
- Definir si caja chica requiere arqueo por turno.
